Global Sales Intelligence Dashboard
An interactive executive-level sales analytics dashboard built using HTML and Claude AI.
What it does:
Analyzes simulated sales data across 4 global regions (EMEA, APAC, NA, LATAM) over 5 years (2014–2018). Includes 12+ dynamic visualizations — revenue trends, discount impact, regional performance, product-wise sales, and a salesperson leaderboard.
Key Features:
KPI cards: Total Revenue, Total Transactions, Avg Discount, Revenue Lost
Dynamic filters: Region, Year, Product Category, Discount Tier
Automated Business Insights panel highlighting top regions, peak periods & recovery opportunities
Clean, professional UI designed for executive reporting
Tools Used:
HTML, CSS, JavaScript, Chart.js, Claude AI
Purpose:
This is a sample/simulated dataset project built to demonstrate business intelligence and data storytelling skills relevant to a Business Analyst role.
